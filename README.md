# rails template

To simply create a new rails app with minimum configuration

```
rails new NAME_OF_YOUR_PROJECT \
          --database=postgresql \
          --skip-yarn \
          --skip-action-cable \
          --skip-coffee \
          --skip-test \
          --skip-system-test \
          -m https://raw.githubusercontent.com/jpheos/rails-template/master/template.rb
```
